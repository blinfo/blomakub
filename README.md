# BL-Omakub


Turn a fresh Ubuntu installation into a fully-configured, beautiful, and modern development system by running a single command. That's the one-line pitch for Omakub.
This is a mod of the original [omakub.org](https://omakub.org) install-script. This installer has(will..) removed a lot of unwanted apps and adds sdkman, mvn, java etc to work with our development env.

Watch the introduction video and read more at [omakub.org](https://omakub.org).
